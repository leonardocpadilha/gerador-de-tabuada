## Sobre o Projeto
Aplicação simples e responsiva desenvolvida para praticar a manipulação do DOM (Document Object Model) com JavaScript puro. O usuário insere um número base e o multiplicador desejado para gerar uma tabela de multiplicação instantaneamente na tela.

O projeto conta ainda com validações para impedir números negativos ou campos vazios, garantindo uma boa experiência de uso.

## Tecnologias Utilizadas
O projeto foi desenvolvido utilizando as seguintes tecnologias:
*   **HTML5** - Estruturação do conteúdo.
*   **CSS3** - Estilização e responsividade.
*   **JavaScript (ES6)** - Lógica de programação, manipulação do DOM e validações.
